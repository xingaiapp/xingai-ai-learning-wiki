# Bilingual ADRs (EN + 中文)

Each ADR in `docs/adr/` ships as **two files**: English (`.md`) and Simplified Chinese (`.zh.md`).

## Naming

| Language | Pattern | Example |
|----------|---------|---------|
| English | `NNN-slug.md` | `001-supervisor-audit-human-in-the-loop.md` |
| 中文 | `NNN-slug.zh.md` | `001-supervisor-audit-human-in-the-loop.zh.md` |

## Section map

| English | 中文 |
|---------|------|
| Context | 背景 |
| Decision | 决策 |
| Alternatives considered | 备选方案 |
| Consequences | 后果 |
| Implementation status | 实现状态 |
| Related | 相关 |
