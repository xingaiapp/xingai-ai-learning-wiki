# XingAI AI Engineering Learning Series

From first principles to CTO-level AI leadership. This bilingual curriculum teaches learners to explain, build, debug, operate, architect, and lead AI systems. It prepares candidates for common **AI engineering hiring loops**; it does not promise a particular hiring outcome.

This is a **documentation curriculum**, not [Learn AI](https://learn.xingai.app) ([`xingai-learn`](https://github.com/xingaiapp/xingai-learn)). Learn AI recommends the next **coding pattern** from a submitted LeetCode-style interview question. This curriculum teaches **AI engineering / architect competencies** through lessons and assessments — system design, tools, RAG, MCP, governance — not DSA pattern drills.

## Required Lesson Contract

Every course follows [COURSE-STANDARD.md](COURSE-STANDARD.md): 5W + How, code, Mermaid diagrams, production failures, a lab, interview questions, and a scored gate. English (`.md`) and Chinese (`.zh.md`) files are complete peers.

## How To Study

1. Take the diagnostic in each course before reading.
2. Complete the code exercise without copying the solution.
3. Explain the diagram aloud at beginner, engineer, architect, and executive depth.
4. Complete the lab and record evidence in a portfolio repository.
5. Pass the course gate at 80% before advancing.
6. Revisit earlier courses from the new seniority level; leadership requires technical recall, not just vocabulary.

## Program Gates

| Gate | Demonstration |
|---|---|
| Understand | Explain concepts, assumptions, and limits accurately |
| Implement | Produce working code with tests and typed boundaries |
| Debug | Isolate model, data, tool, state, security, and runtime failures |
| Design | Compare alternatives and defend tradeoffs with evidence |
| Lead | Connect architecture to users, risk, cost, teams, and outcomes |

See the [assessment framework](../assessments/README.md), [capstones](../capstones/README.md), and [interview bank](../interview-bank/README.md).

## Advanced Track

After completing the role-relevant foundation courses, continue with [Deep Enterprise AI Engineering](../deep-enterprise-ai/README.md). It adds a production-shaped reference POC, enterprise RAG, harness and loop engineering, agent-framework comparison, controlled multi-agent design, MCP, identity/policy, security, observability, audit, evaluation, operations, and CTO architecture defense.
